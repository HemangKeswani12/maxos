# Complete Skincare Stack — Build Guide

## The Golden Rule

**One new product every 4 weeks.** You cannot identify what's helping or causing breakouts if you introduce everything simultaneously.

## Full Protocol (When Fully Ramped)

### Morning
1. **Gentle cleanser** (pH 4.5–6.5)
2. **Vitamin C** (L-ascorbic acid 15–20%) — antioxidant, collagen, brightening
3. **Niacinamide 10%** — sebum, tone, barrier (wait 30s if using with Vitamin C)
4. **Moisturiser** (ceramide-based)
5. **SPF 50+** — the most important step

### Evening
1. **Cleanser** (double cleanse if wearing SPF — oil cleanse first, then water-based)
2. **Active** (choose one per night — see below)
3. **Moisturiser**

## Active Rotation (PM)

| Night | Active | Purpose |
|-------|--------|---------|
| Mon/Thu | Retinoid (adapalene 0.1% or tretinoin) | Anti-aging, acne, texture |
| Tue | Salicylic acid 2% BHA | Comedolytic, pore clearing |
| Wed | AHA 10% (glycolic or lactic) | Exfoliation, brightening |
| Fri | Retinoid | Anti-aging |
| Sat | Niacinamide / rest | Barrier repair |
| Sun | Retinoid | Anti-aging |

Do NOT layer retinoid with BHA or AHA on the same night.

## Layering Order

**Thinnest → Thickest. Water-based → Oil-based.**

1. Cleanser (wash off)
2. Toners / essence (lightest)
3. Serums (actives — water-based)
4. Eye cream (if using)
5. Moisturiser
6. SPF (AM only — always last)

## Skin Type Modifications

### Oily Skin
- Foaming cleanser is fine
- Skip heavy moisturiser in AM — lightweight gel
- Niacinamide priority for sebum control
- BHA (salicylic) is your best exfoliator

### Dry/Sensitive Skin
- Cream cleanser only (no foam)
- Buffer retinoid — apply moisturiser first, then retinoid, then moisturiser again
- Avoid fragrance in all products
- Start AHA/BHA at lower frequency (1×/week)

### Combination
- Zone-specific: Lighter products on T-zone, richer on dry areas
- Niacinamide works for both oil control and barrier

## Introduction Order (For Complete Beginners)

**Month 1:** Cleanser + Moisturiser + SPF only
**Month 2:** Add Niacinamide
**Month 3:** Add Vitamin C
**Month 4:** Add Retinoid (2×/week, titrate)
**Month 5:** Add BHA if acne-prone, or AHA if texture concern
**Month 6+:** Stabilise and optimise

## Product Selection: Ingredient Over Brand

Skip brand loyalty. Read the ingredient list.

**Non-negotiable ingredients to find:**
- Moisturiser: Ceramides (niacinamide, hyaluronic acid as bonus)
- Vitamin C: L-ascorbic acid (not ascorbyl glucoside — weaker)
- Retinoid: Tretinoin (Rx) or adapalene 0.1% (OTC)
- SPF: PA+++ or PPD rating alongside SPF50 (UVA protection matters as much as UVB)
